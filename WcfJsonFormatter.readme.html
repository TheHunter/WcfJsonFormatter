Welcome to WcfJsonFormatter

WcfJsonFormatter is a simple library which contains a set of basic implementations for managing Json messages formatters.

Latest Version

The quickest way to get the latest release of WcfJsonFormatter is to add it to your project using NuGet (http://nuget.org/List/Packages/WcfJsonFormatter).
